YouTube Watch Party App

A real-time YouTube watch party application where multiple users can join a room and watch videos together in sync.

---

Live Demo

👉 Frontend: (https://watch-party-eq9z.vercel.app)
👉 Backend: (https://watch-party-7qe9.onrender.com)

---

##  Tech Stack

* React.js
* Node.js
* Express.js
* Socket.IO

---

##  Features

* Create / Join Room
* Real-time Video Sync
* Chat System
* Multiple Users Support

---

##  Installation

### Clone Project

```bash
git clone https://github.com/your-username/your-repo-name.git
```

### Install Dependencies

```bash
cd client
npm install

cd ../server
npm install
```

### Run Project

```bash
# backend
cd server
npm start

# frontend
cd client
npm start
```

---

##  Architecture

* Frontend (React) handles UI
* Backend (Node + Express) handles API
* Socket.IO handles real-time communication
* Users join rooms and sync video events
